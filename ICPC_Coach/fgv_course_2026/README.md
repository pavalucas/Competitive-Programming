# FGV ICPC Preparation Course 2026

This folder contains the material for the lectures I am giving at **FGV (Fundacao Getulio Vargas)** to help students prepare for the **ICPC** and for competitive programming contests in general.

The goal of the course is to build the core problem-solving toolkit required in programming competitions.

## Course Structure

Each subfolder groups problems and solutions by topic. The intention is to combine:

- short theoretical explanations during class
- discussion of common patterns and mistakes
- implementation practice in C++
- post-class problem solving

## Current Topics

### [`data_structures_basic`](./data_structures_basic)

Introductory problems focused on basic data structures and common contest patterns, including:

- maps and frequency counting
- sets and registration-style problems

## Target Audience

This material is intended for:

- FGV students starting in competitive programming
- students preparing for the ICPC selection process
- teams looking to strengthen fundamentals before moving to advanced topics

## Language

The solutions in this folder are written in **C++**, which is the standard language used throughout the course.

## Repository Usage

This directory is meant to serve as both:

- a lecture companion during the course
- a practice archive for students reviewing problems after class

Students are encouraged to:

- solve each problem before reading the solution
- reimplement key ideas without looking at the code
- revisit the same topic until the patterns become natural
